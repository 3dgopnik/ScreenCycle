# Быстрый старт

1. Склонируйте репозиторий и откройте папку в **Android Studio** (Arctic Fox или новее).
2. Проверьте, что файл `gradle.properties` содержит:
   ```
   android.useAndroidX=true
   android.enableJetifier=true
   ```
   Эти флаги включают AndroidX и Jetifier.
3. Подключите устройство Android или запустите эмулятор.
4. Нажмите **Run > Run 'app'** или выполните в терминале:
   ```bash
   ./gradlew installDebug
   ```
5. После установки откройте приложение на устройстве.

Ожидаемый результат: приложение запускается и отображает главный экран без ошибок.
